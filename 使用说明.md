# HF 动作编辑器 使用说明

Hero Fighter（HFE v1.0.2 / HF-EX v0.2.5）角色动作与贴图的可视化编辑工具。
在浏览器里直接读写 HFWorkshop 导出的数据文件夹，实时预览动作效果，无需反复导入游戏。

## 环境要求

- **Edge 或 Chrome 浏览器**（需要"文件夹读写"能力，其他浏览器不支持）
- 无需安装任何东西：双击 `index.html` 即可打开

## 准备数据

1. 用 **HFWorkshop** 从游戏 exe 导出角色的两个数据：
   - `XXX - Data.Global_角色名Spt`（角色数据，含 Spt.json）
   - `YYY - Data.Global_角色名Lmi`（贴图数据，含 Limb_*.json / LimbPic_*.json / *.png）
2. 解压到**同一个文件夹**里（比如一个叫 `我的角色` 的文件夹，里面放这两个子文件夹）

## 基本流程

1. 双击 `index.html` 打开工具（用 Edge/Chrome）
2. 点 **📂 打开角色** → 选择上面准备的文件夹 → 浏览器询问权限时点"编辑文件"
3. 左侧选动作 → 底部时间轴选帧 → 中间画布编辑
4. **💾 保存**（Ctrl+S）：直接写回文件夹里的 JSON
5. 把两个文件夹分别压成 zip（或直接用 **📦 导出 zip**），用 HFWorkshop 导入回游戏

## 画布操作

| 操作 | 方式 |
|---|---|
| 缩放视图 | 鼠标滚轮 |
| 平移视图 | 按住中键/右键拖动 |
| 选择部位 | 左键点击角色的某个部位（按像素精确命中） |
| 旋转部位 | 工具条选 **⟳ 旋转** → 拖拽选中的部位（子部位会跟着转，Shift=15° 吸附） |
| 移动部位 | 工具条选 **✥ 移动** → 拖拽（调整 dp 偏移；拖"胸"=整体移动） |
| 缩放部位 | 工具条选 **⤢ 缩放** → 拖拽（离关节越远越大；Shift=仅纵向） |
| 播放动画 | 空格键 或 ▶ 按钮；`←` `→` 逐帧切换 |
| 撤销/重做 | Ctrl+Z / Ctrl+Y |

**洋葱皮**：编辑当前帧时半透明叠显前后帧，方便做连贯动作。
**判定框**：显示攻击框（红）/受击框（绿）/默认受击框（蓝）。

## 右侧面板

- **部位**：绘制顺序列表（上=顶层，▲▼调层级）、旋转/缩放/位移数值、贴图造型切换
  - 标灰的部位是"联动从属"（如护腕随前臂），编辑主部位即可
- **帧**：duration（停留 tick）、移动速度 vx/vy、音效、footY（可自动重算）、末帧标志等
- **判定框**：攻击框/受击框的新增、删除、数值编辑（锚定在部位关节上，跟随姿势移动）
- **动作**：动作名、自循环、结束/落地跳转、耗蓝等
- **贴图**：
  - **替换PNG**：给某张部位贴图换皮肤（锚点关节保持不变）
  - **导入PNG新增变体**：给部位加新贴图造型，自动生成 LimbPic json 并更新索引
  - **锚点/关节编辑**：拖拽调整挂接点（j0红=挂父级的点，j1+绿=挂子部位的点），可叠影参照现有贴图

## 帧与动作管理

- **＋复制帧**：复制当前帧插到后面（复制末帧时动作自动延长）
- **－删除帧**：删除当前帧（每个动作至少留一帧；引用了该帧的其他帧会自动清除引用并提示）
- **复制动作**（左栏顶部）：把整个动作连同全部帧复制为新动作，用来做新招式
  - 注意：新动作还需要在按键触发（keyTgr）或跳转里挂接才能在游戏里触发，目前请在"动作"页配置跳转，或用其他工具配置按键

## 保存与导入游戏

- **💾 保存**只写有改动的文件（未动过的文件保证逐字节不变）
- **📦 导出 zip** 会生成 `XXX - ...Spt.zip` 和 `YYY - ...Lmi.zip` 两个压缩包，直接用 HFWorkshop 导入
- **🩺 兼容性体检**：校验工具对文件的解析-还原是否逐字节一致，任何时候不放心可以点一下

## 注意事项

1. **编辑前备份原始导出文件夹**（工具直接写回文件夹）
2. 特效（模糊、发光、冰火着色）在编辑器里是**近似预览**，以游戏内实际效果为准
3. 修改姿势后工具会自动重算：裁剪框（cx1..cy2）、footY（可关）、运行时判定框、lp1/lp3——这些都是游戏需要的一致性数据，不需要手动管
4. 带"引用外部姿势 rsn"标记的帧来自模型精灵（model0），编辑它不会生效，请先复制帧
5. HFE 和 HFEX 的数据都支持，打开时自动识别（右上角有版本徽章），新建的数据自动匹配对应版本的字段

## 常见问题

**Q：打开文件夹后提示"未找到 Spt 文件夹"？**
A：确认选的是包含 `* - Data.Global_*Spt` 和 `* - Data.Global_*Lmi` 两个子文件夹的那一层。

**Q：画布上角色显示不全/贴图缺失？**
A：等待 1-2 秒（PNG 异步解码）；若仍缺，检查 Lmi 文件夹的 PNG 是否齐全。

**Q：改了姿势游戏里没变化？**
A：确认保存后重新压缩并用 HFWorkshop 导入了正确的数据条目（Spt 对应角色数据编号）。
